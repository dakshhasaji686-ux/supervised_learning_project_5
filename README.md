# 🎓 Smart Outcome Predictor using Ensemble Learning

## 📌 Overview
This project predicts **student course completion (classification)** and **final scores (regression)** using Ensemble Learning algorithms. It compares multiple ensemble techniques to identify the best-performing model.

## 🚀 Algorithms Used
- Bagging
- AdaBoost
- Gradient Boosting
- LightGBM
- XGBoost
- Voting Ensemble
- Stacking Ensemble

## 🛠 Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- LightGBM
- XGBoost
- Matplotlib

## 📊 Evaluation Metrics

**Classification**
- Accuracy
- Precision
- Recall
- F1-Score
- ROC-AUC

**Regression**
- MAE
- RMSE
- R² Score


## 🎯 Objective
- Predict course completion status.
- Predict students' final scores.
- Compare ensemble learning techniques.
- Recommend the best model for deployment.
